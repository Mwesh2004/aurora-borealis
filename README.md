🌌 Aurora Borealis Chrome Theme

A calm, immersive browser experience inspired by the northern lights.

✨ Overview

Aurora Borealis is a carefully crafted Chrome theme designed to transform your browsing experience into something atmospheric, modern, and visually soothing. Inspired by the natural phenomenon of the northern lights, this theme blends deep dark tones with glowing aurora gradients to create a balanced interface that reduces eye strain while enhancing aesthetic appeal.

It is built for users who spend long hours in the browser and want a cleaner, more focused, and visually refined environment.

🌠 Inspiration

The Aurora Borealis (northern lights) is one of nature’s most beautiful light displays — flowing waves of green, cyan, violet, and icy blue across the night sky.

This theme captures that feeling digitally:

Soft glowing gradients
Subtle atmospheric depth
Smooth visual transitions
Calm, immersive dark mode foundation

The result is a browser that feels less like a tool and more like an experience.

🎨 Design Philosophy

Aurora Borealis is built on three core principles:

🧘 Calmness

Reduce visual noise and create a distraction-free browsing environment.

🌌 Depth

Use layered dark tones and glow effects to create a sense of immersion.

⚡ Clarity

Maintain readability and contrast across all UI elements without overwhelming brightness.

✨ Key Features

🌙 Deep Dark Base UI
A rich, low-light foundation optimised for long browsing sessions and reduced eye fatigue.

💫 Aurora Gradient Accents
Beautiful blends of cyan, green, violet, and ice-blue inspired by real aurora motion.

🧊 Minimal Visual Clutter
Clean and subtle UI styling that keeps focus on content, not interface noise.

🌊 Soft Glow Aesthetic
Gentle lighting effects that add depth without overwhelming brightness.

🧠 Focus-Oriented Design
Designed to improve concentration during work, study, coding, or research sessions.

⚡ Modern Browser Experience
A sleek and contemporary look that pairs well with productivity and creative workflows.

Manual Installation (Developer Mode)
Download or clone this repository
Open Chrome → Extensions
Enable Developer Mode
Click Load Unpacked
Select the project folder
Theme will be applied instantly
